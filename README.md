#  Expense Tracker (Java)

A **console-based Expense Tracker application** built using **Java**, designed to help users record, categorize, and track their daily expenses.  
This project demonstrates **core Java fundamentals, Object-Oriented Programming (OOP), and menu-driven application design**.

---

##  Features

-  Add daily expenses  
-  Categorize expenses (food, travel, etc.)  
-  View all recorded expenses  
-  Calculate total expenses  
-  Menu-driven console interface  

---

##  Technologies Used

- **Language:** Java  
- **Concepts Used:**  
  - Object-Oriented Programming (OOP)  
  - Classes and Objects  
  - Encapsulation  
  - Conditional Statements  
  - Loops  
  - Arrays / ArrayList  
- **Utilities:** Scanner (for user input)

User is presented with the expense tracker menu
User selects an operation
Program records or displays expense data
Menu repeats until the user exits
